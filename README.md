# Train Dataset Analysis using Dask

## Project Overview
This project analyzes the BigMart sales dataset (Train.csv) using Python and Dask for large-scale data processing.

## Dataset
Train.csv  
Total records: 8523

## Analysis Performed
- Loaded dataset using Dask
- Displayed first 5 rows of data
- Calculated total number of rows
- Computed average Item MRP
- Calculated total sales by outlet type
- Calculated average sales by item type

## Key Insights
- Supermarket Type1 has the highest total sales.
- Starchy Foods category has the highest average sales.

## Tools Used
Python  
Dask
